# 🛡️ Proyecto: Seguridad en el Trabajo Informático

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Desplegado-success?style=for-the-badge&logo=github)](https://[tu-usuario].github.io/seguridad-trabajo-informatica/)
[![Licencia](https://img.shields.io/badge/Licencia-Creative%20Commons-blue?style=for-the-badge)](https://creativecommons.org/licenses/by/4.0/)

Este repositorio contiene un proyecto de investigación detallado sobre la **prevención de riesgos laborales y ergonomía** en el sector tecnológico, desarrollado para la asignatura de **Informática**. El sitio utiliza **Jekyll** y está desplegado mediante **GitHub Pages** con un enfoque en la legibilidad y la salud visual.

## 🚀 Características Avanzadas (Markdown & CSS)
Para elevar la calidad del proyecto, hemos implementado soluciones técnicas de alto nivel:
* **Ergonomía Digital:** Interfaz diseñada con tipografías de alta legibilidad y espaciados calculados para reducir la fatiga visual.
* **CSS Custom Properties:** Gestión de colores mediante variables para asegurar un contraste óptimo (WCAG 2.1).
* **Componentes Dinámicos:** Menú de navegación con efectos `hover` y transiciones suaves para mejorar la experiencia de usuario (UX).
* **Tablas de Datos:** Diseño de tablas cebra estilizadas para la comparativa de riesgos y soluciones.
* **Layouts con Jekyll:** Uso de plantillas `default.html` para mantener la consistencia en todas las páginas.

## ⚙️ Instrucciones de Instalación
Para previsualizar este proyecto en un entorno de desarrollo local:
1. Clona el repositorio: `git clone https://github.com/[tu-usuario]/seguridad-trabajo-informatica.git`
2. Instala las dependencias: `bundle install`
3. Ejecuta el servidor: `bundle exec jekyll serve`
4. Accede en tu navegador a: `http://localhost:4000`
